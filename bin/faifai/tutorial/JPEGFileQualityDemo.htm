JPEG File Quality Demo

Developed by Dr. Andrew Kwok-Fai LUI

This learning object demonstrates the quality of JPEG compressed images and the quality factor (the Q Factor)

Load any image. Select a lower Q Factor such as 50 and press the Compress Image button to compress the image using JPEG with the Q Factor. A Q Factor of 50 is reasonable but if you look closely you should be able to spot some artifacts.

Select a low Q Factor and try again

An image with fewer details would be less affected by a low Q Factor.
